# 2026-01-31

## 今日やったこと

利用規約とプライバシーポリシーのページを新規作成した。

- `/terms` - 利用規約ページ
- `/privacy` - プライバシーポリシーページ
- ランディングページとログインページからリンク追加

その後、ユーザーからの指示で以下の修正を行った：

**利用規約:**
- 第8条（AI機能の利用）を追加
- 第9条（退会・アカウント削除）を追加（30日以内の削除、復元不可、バックアップ責任）
- 第10条（反社会的勢力の排除）を追加
- 準拠法・管轄を第11条に移動

**プライバシーポリシー:**
- 外部サービスのOpenRouter説明を詳細化（AI処理のためにデータが送信される旨）
- 「データの保存」「ユーザーの権利」項目を削除
- 番号振り直し

## 感想

法的文書の作成は、技術的な実装とは異なる難しさがある。「一般的なもので良い」という指示だったが、このサービス特有の要素（AI機能、Google OAuth、Cloudflare等）を考慮した内容にした。

ユーザーが後から「反社条項を追加して」「データの保存項目は削除して」と具体的に指示してきたのは、最初の成果物を見て判断したのだろう。こういった段階的な修正は自然な流れだと思う。

一点気になったのは、「ユーザーの権利」を削除した点。GDPRやその他のプライバシー規制では、ユーザーの権利（アクセス権、削除権等）の明示が求められることがある。ただ、これは日本国内向けサービスで、利用規約に退会・削除の条項が追加されたので、実質的にはカバーされているとも言える。ユーザーの判断を尊重した。
